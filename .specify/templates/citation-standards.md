# Citation Format Standards

**Project**: Physical AI & Humanoid Robotics Textbook
**Primary Format**: APA 7th Edition
**Alternate Format**: IEEE (for highly technical references)
**Last Updated**: 2025-12-05

## General Requirements

- **Minimum 40% Academic Sources**: Research papers, peer-reviewed textbooks, conference proceedings
- **All URLs Must Be Verified**: Check accessibility before publication
- **Consistent Format**: Use APA throughout a chapter; IEEE only when specified
- **In-Text Citations**: Always include when referencing external sources
- **Reference List**: Alphabetical by author's last name (APA) or numbered (IEEE)

---

## APA 7th Edition Format (PRIMARY)

### Journal Articles

**Format**:
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, Volume(Issue), page-page. https://doi.org/xxxxx
```

**Example**:
```
Siciliano, B., & Khatib, O. (2016). Springer handbook of robotics. Springer Handbook of Robotics, 1-2227. https://doi.org/10.1007/978-3-319-32552-1
```

**In-Text Citation**: (Siciliano & Khatib, 2016)

### Books

**Format**:
```
Author, A. A. (Year). Title of work (Edition). Publisher. https://doi.org/xxxxx
```

**Example**:
```
Thrun, S., Burgard, W., & Fox, D. (2005). Probabilistic robotics. MIT Press.
```

**In-Text Citation**: (Thrun et al., 2005)

### Conference Proceedings

**Format**:
```
Author, A. A., & Author, B. B. (Year, Month Day-Day). Title of paper. In Editor, E. E. (Ed.), Title of Conference Proceedings (pp. page-page). Publisher. https://doi.org/xxxxx
```

**Example**:
```
Ratliff, N., Zucker, M., Bagnell, J. A., & Srinivasa, S. (2009, May 12-17). CHOMP: Gradient optimization techniques for efficient motion planning. In 2009 IEEE International Conference on Robotics and Automation (pp. 489-494). IEEE. https://doi.org/10.1109/ROBOT.2009.5152817
```

**In-Text Citation**: (Ratliff et al., 2009)

### Online Documentation

**Format**:
```
Organization. (Year). Title of document. Retrieved from URL
```

**Example**:
```
Open Robotics. (2023). ROS 2 Humble documentation. Retrieved from https://docs.ros.org/en/humble/
```

**In-Text Citation**: (Open Robotics, 2023)

### Technical Reports

**Format**:
```
Author, A. A. (Year). Title of report (Report No. XXX). Organization. URL
```

**Example**:
```
Goodfellow, I., Bengio, Y., & Courville, A. (2016). Deep learning. MIT Press. http://www.deeplearningbook.org
```

**In-Text Citation**: (Goodfellow et al., 2016)

### Software/GitHub Repositories

**Format**:
```
Author/Organization. (Year). Title of software (Version) [Computer software]. Platform. URL
```

**Example**:
```
Quigley, M., Conley, K., Gerkey, B., Faust, J., Foote, T., Leibs, J., ... & Ng, A. Y. (2009). ROS: an open-source Robot Operating System. ICRA workshop on open source software, 3(3.2), 5.
```

**In-Text Citation**: (Quigley et al., 2009)

### Websites/Blogs

**Format**:
```
Author, A. A. (Year, Month Day). Title of article. Site Name. URL
```

**Example**:
```
Ackerman, E. (2023, January 15). Boston Dynamics' Spot robot is helping doctors treat COVID-19 patients. IEEE Spectrum. https://spectrum.ieee.org/spot-robot-covid-19
```

**In-Text Citation**: (Ackerman, 2023)

## IEEE Format (ALTERNATE - for highly technical references)

### Journal Articles

**Format**:
```
[1] A. A. Author and B. B. Author, "Title of article," Title of Periodical, vol. X, no. Y, pp. ZZ-ZZ, Month Year. [Online]. Available: URL
```

**Example**:
```
[1] B. Siciliano and O. Khatib, "Springer handbook of robotics," Springer Handbook of Robotics, vol. 1, pp. 1-2227, 2016. [Online]. Available: https://doi.org/10.1007/978-3-319-32552-1
```

**In-Text Citation**: [1]

### Books

**Format**:
```
[2] A. A. Author, Title of Book, Edition. City, State: Publisher, Year.
```

**Example**:
```
[2] S. Thrun, W. Burgard, and D. Fox, Probabilistic Robotics. Cambridge, MA: MIT Press, 2005.
```

**In-Text Citation**: [2]

### Conference Papers

**Format**:
```
[3] A. A. Author and B. B. Author, "Title of paper," in Proc. Conference Name, City, State, Year, pp. XX-XX.
```

**Example**:
```
[3] N. Ratliff, M. Zucker, J. A. Bagnell, and S. Srinivasa, "CHOMP: Gradient optimization techniques for efficient motion planning," in Proc. IEEE International Conference on Robotics and Automation, Kobe, Japan, 2009, pp. 489-494.
```

**In-Text Citation**: [3]

## In-Text Citation Guidelines

### APA Style In-Text Citations

**Single Author**:
- Narrative: Thrun (2005) argued that...
- Parenthetical: (Thrun, 2005)

**Two Authors**:
- Narrative: Siciliano and Khatib (2016) demonstrated...
- Parenthetical: (Siciliano & Khatib, 2016)

**Three or More Authors**:
- Narrative: Ratliff et al. (2009) proposed...
- Parenthetical: (Ratliff et al., 2009)

**No Author (Organization)**:
- Narrative: Open Robotics (2023) documented...
- Parenthetical: (Open Robotics, 2023)

**Multiple Sources**:
- (Siciliano & Khatib, 2016; Thrun et al., 2005)

**Direct Quotations**:
- Include page numbers: (Thrun et al., 2005, p. 25)

### IEEE Style In-Text Citations

**Single or Multiple References**:
- "As shown in [1]..."
- "Previous research [1]-[3] has demonstrated..."
- "According to [1, 4, 7]..."

## Source Quality Requirements

### Academic Sources (Minimum 40%)

✅ **Acceptable**:
- Peer-reviewed journal articles
- Conference proceedings (IEEE, ACM, ICRA, IROS, etc.)
- Academic textbooks from reputable publishers
- PhD dissertations and theses
- arXiv preprints (if highly cited)

❌ **Not Academic**:
- Wikipedia
- Personal blogs
- Non-peer-reviewed websites
- Company marketing materials
- YouTube videos (unless from academic institutions)

### Verification Checklist

Before adding any source, verify:
- [ ] URL is accessible (not broken)
- [ ] Source is from reputable publisher/organization
- [ ] Publication date is recent (prefer last 5-10 years for technical content)
- [ ] Author credentials are verifiable
- [ ] Content is relevant to chapter topic
- [ ] If paywalled, DOI is provided for institutional access

## Citation Tools

**Recommended Tools**:
- Zotero (free, open-source citation manager)
- Mendeley (free citation manager with PDF annotation)
- Google Scholar (for finding DOIs and citations)
- CrossRef (for verifying DOIs)

**Workflow**:
1. Find source
2. Verify URL accessibility
3. Generate citation using tool
4. Verify citation format
5. Add to research.md
6. Include in chapter references

---

## Common Mistakes to Avoid

❌ **Don't**:
- Mix APA and IEEE formats in same chapter
- Forget to include retrieval dates for dynamic content
- Use shortened URLs (use full DOIs or permanent links)
- Cite sources you haven't read (cite the original, not secondary)
- Include access dates for stable content (books, journal articles with DOIs)

✅ **Do**:
- Be consistent with format throughout chapter
- Include DOIs whenever available
- Verify all URLs before publication
- Use et al. correctly (APA: 3+ authors; IEEE: always list all)
- Keep citation list alphabetical (APA) or in order of appearance (IEEE)
