### Technology Stack

This textbook teaches you to work with cutting-edge robotics technologies:

- **ROS 2 (Robot Operating System)**: Industry-standard middleware for robot software
- **Gazebo**: High-fidelity physics simulation for safe testing
- **OpenCV**: Computer vision library for image processing
- **OpenAI Whisper**: State-of-the-art speech recognition
- **YOLO**: Real-time object detection
- **BehaviorTree.CPP**: Task planning and execution framework
- **MoveIt 2**: Motion planning for manipulation
- **Python & C++**: Primary programming languages

### Learning Philosophy

Our approach is built on three pillars:

1. **Learn by Building**: Theory is important, but nothing beats hands-on experience. You'll write code, debug issues, and see results immediately.

2. **Fail Fast, Learn Faster**: Simulation lets you experiment without consequences. Break things, try new approaches, and iterate quickly.

3. **Integration Over Isolation**: Real robotics requires integrating perception, planning, and control. Each module prepares you for the final integration project.
